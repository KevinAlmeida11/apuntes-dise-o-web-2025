Elementos con significado!

Ejemplo de etiquetas no semánticas
Ejemplo de etiquetas semánticas:Table,Form,Article

Header

Es el encabezado de una página,sección o artículo.Incluye elementos como el logo,texto ontroductorio,menus de navegacion.Puede haber multiples headers en una página,pero no puede haber un header dentro de un footer,o de otro header.

footer

Es el pie de página de un sitio o sección.Puede haber múltiples por página.Puede incluir información como:
  • Información de contacto
  • Enlaces a las RRSS
  • Mapa del sitio(sitemap)
  • Créditos
  • Formulario de Contacto

Nav

Define un conjunto d enlaces de navegación.No todos los links deebn estar dentro de un "Nav".

Main

Crea un contenedor para el contenido principal de nuestra página.Este debe ser único y no debe incluir contenido repetido de otras secciones,como por ej del sidebar,logos,etc.No puede ser hijo de estiquetas como: article,aside,footer,header o nav.

Aside

Define el contenido relacionado indirectamente con el contenio principal como por ejemplo un sidebar.Elemento que solemos poner en un aside:
  • Publicidad
  • Biografía del autor
  • Enlaces relacionados
  • Información de Perfil

Section

Define una sección de una página donde se agrupa los contenidos relacionados,y normalmente incluyen un título.
Una web se compone de varias secciones,cada una con su cpontenido y propósito.
Ejemplo de uso:
  • Capítulos
  • Introducción
  • Noticias
  • Información de contacto

Article

Define un contenido INDEPENDIENTE.El mismo puede ser compartido o reutilizado en otro sitio sin requerir contexto.
Ejemplos:
  • Un post de un blog
  • Un artículo de una revista
  • Comentario de un usuario(Reseña)
  • Tarjeta de producto

Puede encontrar artículos dentro de secciones,y secciones dentro de artículos. Un artículo puede tener múltiples secciones.Una sección puede tener múltiples artículos.

Figure

Define elementos autocontenidos,como puede ser una imágen,un diagrama,un gráfico,una tabla o una foto.
Dentro de ella,se suele utilizar la etiqueta "figcaption" para describir el contenido,esto es además del ya incluido "alt" en caso de imágenes.

Mark

Define un texto resaltado,normalmente se muestra con fondo amarillo
