## Markdown
Es un lenguaje de marcado ligero que se utiliza para dar formato a un texto plano.

Headings:
# Título 1 (igual a h1)
## Título 2 (igual a h2)
### Título 3 (igual a h3)
#### Título 4 (igual a h4)
##### Título 5 (igual a h5)
###### Título 6 (igual a h6)



Emphasis
* texto en Italic* o _Texto en Italic_
**Texto en negrita** __texto en negrita__


Liatas:

- Item 1
- Item 2
- Item 3

1. Item 1
2. Item 2
3. Item 3
4. item 4
    - subitem 1
    - subitem 2
    - subitem 3
